Инструкция по запуску проекта. Для запуска проекта на компьютере должны быть: .net8, asp.net, react, node.js 
Скачиваем себе проект и открывает решение. 
Ждём, пока все пакеты npm подкачаются. 
Когда установка завершилась, открываем в терминале папку LibraryWebApplication.client и в ней прописываем: npm install npm init -y npm run build. Затем запускаем решение(сервер)
